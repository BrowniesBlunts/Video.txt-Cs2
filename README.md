# video.txt – Configuración de Video en Counter-Strike 2

## Introducción

El archivo `video.txt` en **Counter-Strike 2 (CS2)** es un archivo de configuración que almacena los parámetros gráficos del juego. A diferencia de otros ajustes realizados desde el menú, este archivo permite definir de forma directa opciones relacionadas con resolución, modo de pantalla, escalado y renderizado.

El uso correcto del archivo `video.txt` es fundamental para optimizar el rendimiento, reducir latencia visual y mantener consistencia gráfica entre sesiones.

## Ubicación del archivo

El archivo `video.txt` se encuentra generalmente en la siguiente ruta:

Steam\userdata<SteamID>\730\local\cfg\


Este archivo se genera automáticamente al modificar opciones gráficas desde el juego.

---

## Función del archivo video.txt

El archivo `video.txt` guarda configuraciones relacionadas con:

- Resolución de pantalla.
- Modo de pantalla (fullscreen, windowed, borderless).
- Escalado de imagen.
- Uso de GPU y renderizado.
- Sincronización vertical (V-Sync).
- Opciones avanzadas de video.

Estas configuraciones influyen directamente en los FPS, input lag y claridad visual.

---

## Importancia en el rendimiento

Una configuración correcta del `video.txt` permite:

- Mejor estabilidad de FPS.
- Reducción de latencia visual.
- Menor carga innecesaria de GPU.
- Experiencia más fluida en escenarios competitivos.

Por esta razón, muchos jugadores competitivos ajustan este archivo manualmente.

---

## Ventajas del uso manual del video.txt

- Control total sobre los parámetros gráficos.
- Consistencia entre diferentes equipos.
- Posibilidad de aplicar ajustes no siempre expuestos en el menú.
- Fácil respaldo y restauración.

---

## Riesgos y consideraciones

- Modificar valores incorrectos puede causar errores gráficos.
- Algunas configuraciones se sobrescriben al cambiar opciones en el menú.
- Cambios incompatibles con el hardware pueden provocar inestabilidad.

Se recomienda realizar copias de seguridad antes de editar el archivo.

---

## Relación con otros archivos de configuración

El archivo `video.txt` trabaja junto a:

- `autoexec.cfg` (configuración general).
- `config.cfg` (ajustes del usuario).
- Parámetros de lanzamiento de Steam.

Todos forman parte del ecosistema de configuración avanzada de CS2.

El archivo `video.txt` es una pieza clave en la optimización gráfica de Counter-Strike 2. Su correcta configuración permite mejorar el rendimiento, reducir distracciones visuales y mantener una experiencia competitiva estable. El conocimiento y manejo de este archivo demuestra un entendimiento técnico avanzado del juego.
